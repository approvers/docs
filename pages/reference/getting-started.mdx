# はじめに

限界開発鯖に参加したら, まずはこのドキュメントを読んでください.

## メンバーデータの追加

限界開発鯖では各メンバーの情報をデータベースに保存し, それらを元に様々な機能を提供しています.

限界開発鯖に参加したら, まずは自分のメンバーデータを追加してください.

1. [members.approvers.dev](https://members.approvers.dev/) にアクセスします.
2. `アカウント情報を登録する` をクリックします.
3. ご自身の Discord アカウントと `members-assoc` の App を連携します.

### メンバーデータの編集

参加後, メンバーデータを編集したい場合は以下の手順を行ってください.

1. [edit.members.approvers.dev](https://edit.members.approvers.dev/) にアクセスします.
2. `Discord でログイン` をクリックし, ご自身の Discord アカウントにログインします.
3. X(旧Twitter) ・ GitHub のアカウントを編集します.
4. `保存` をクリックします.

## 自己紹介

参加したら, `#自己紹介` でメンバーのみんなに自己紹介しましょう.

限界開発鯖には高専生じゃない人もいるので, どんな人なのかを知ってもらうためにも自己紹介することを推奨します.

書くことに迷った人用に自己紹介する際のテンプレートをここに書いておきます.

```markdown
名前:
所属:
できること:
興味があること:
X:
GitHub:
```

## Bot の導入

限界開発鯖では自分が開発した Bot を導入できます.

詳しくは [Botガイドライン](../guideline/bot-create-guideline) をご覧ください.

## times の作成

限界開発鯖では各メンバーが個人のチャンネルを持ち, 発言できる場 **times** を設けています.

times はカテゴリー【†限界times†】上に展開しています. 自由にチャンネルを作成してください.
