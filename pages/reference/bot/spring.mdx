# エライさんv2

みんなの「えらい！」をカウントしてくれる Discord Bot.

> あなたのえらさを表す「✨えらいポイント」、つらさを表す「💊よしよしポイント」を集めるBOTです。

- [リポジトリ: 2RiniaR/spring-server](https://github.com/2RiniaR/spring-server)

## 人を 「褒める」

相手のメッセージに以下のリアクションを送信すると **褒める** ことができます.

褒めると自分と相手にえらいポイント`✨`が `1` ずつ加算されます.

- 👏
- 👍
- 🎊
- 🎉
- 🙌
- ✨
- 💪
- ⭐

## 人を「慰める」

相手のメッセージに以下のリアクションを送信すると **慰める** ことができます.

褒めると自分にえらいポイント`✨` が, 相手によしよしポイント`💊` が `1` ずつ加算されます.

## 「生きる」

その日初めてのメッセージを送信すると, その日の生存報告としてえらいポイント`✨` が `1` 加算されます.

## 「健康な睡眠を送る」

`18:00 ~ 01:00` の間に **おやすみ** と送信し, `04:00 ~ 11:00` に何かしらのメッセージを送信すると健康な睡眠を取ったとしてえらいポイント`✨` が `1` 加算されます.

ただし, おやすみを送信した後に `04:00 ~ 11:00` にメッセージを送信しなかった場合・その間にメッセージを送信した場合は, えらいポイント`✨` は加算されません.

## コマンド

すべてのコマンドは `!erai` が Prefix です.

### `!erai user [userId]`

指定したユーザーのステータスを表示します.

### `!erai me`

自分のステータスを表示します.

### `!erai ranking`

えらいポイント`✨` のランキングを表示します.

### `!erai help`

ヘルプを表示します.
